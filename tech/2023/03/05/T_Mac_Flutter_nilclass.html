맥북에서 undefined method `map' for nil:NilClass 오류 발생 시

2023년 3월 5일
  • tech
  • programming
  • flutter
  • error

최근 공부 겸 토이프로젝트 진행 중인데 매번 안드로이드 에뮬레이터에서만 하다가
이번엔 iOS 시뮬레이터에서 해볼까? 하고 진행 중..

이것 저것 테스트 해본다고 여러 패키지들과 코드가 난잡하게 있던 상황에서
빌드를 진행했는데 갑자기 .. 오류가 발생했다.

Searching for inspections failed: undefined method `map' for nil:NilClass

방금 전까지 분명 잘 되던 상태였는데 갑자기 오류가 발생하니 당황 ..
테스트한다고 했던 코드랑 패키지를 하나씩 제거하다보니 역시나 특정 패키지에서
문제가 있었던 것이고 해당 패키지는 사용하고 싶어서 해결 방법을 찾아봤다.

찾아보니 M1 맥북이어서 발생하는 문제로 보였다.
로제타 관련 이슈인 것 같아보였다. StackOverflow에서 관련 글을 확인했고 해당 글은
여기이다.

답변에서 제시한 방법으로 하니 당연히 동작은 되었고 관련해서 찾다보니
Flutter에서도 나름대로 내용을 공유해놓고 있었다.

https://docs.flutter.dev/get-started/install/macos#deploy-to-ios-devices

sudo gem uninstall ffi && sudo gem install ffi -- --enable-libffi-alloc
