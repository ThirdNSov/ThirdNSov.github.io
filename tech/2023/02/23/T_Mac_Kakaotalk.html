맥북에서 카톡 알림 안울릴 때

2023년 2월 23일
  • tech
  • mac
  • macbook
  • kakao
  • alarm

맥북을 구매한지 이제 반년이 조금 지났는데 초반에 카톡 알림이 울리지 않아서
찾아보았던 결과를 기록하려고 한다.

구글, 네이버 검색으로 얻은 해결 방법으로는 해결하지 못했었다.

  1. 카카오톡 설정 확인 (해결 안됨) - 카카오톡 -> 환경설정 -> 알림
  1. 맥북 시스템 설정 확인 (해결 안됨) - 시스템 설정 -> 알림 -> 카카오톡
  2. 방해 금지 모드 설정 해제 확인 (해결 안됨) - 시스템 설정 -> 집중 모드 -> 방해금지 모드

검색해서 확인한 모든 블로그에서 제시하는 방법은 위 세가지인데
둘 다 정상적으로 설정이 되어있는 상태였다.

재설치도 해보고 재설정도 해보고 해봤지만 해결되지 않아서 포기하려던
찰나 우연히 네이버 카페(맥쓰사)를 발견하게 되었고
카페 내에서 검색하다가 어떤 댓글을 보고 해결하게 되었다.

해결방법을 제시했던 댓글은 아니고 그냥 힌트? 정도의 느낌이었다.
찾아본지 시간이 오래 지나서 어떤 게시글의 어떤 내용이었는지 정확히 기억나진 않지만
해당 댓글에 딱 이 단어가 있었던 것은 기억난다.

외장 모니터

그냥 지나칠 수도 있었지만 당시 맥북에 모니터 두대를 연결해서 사용하는 구성이었고
지푸라기라도 잡고 싶은 심정으로 외장모니터 케이블을 다 뺀 후에 확인을 해보았다.

거짓말 처럼 카톡 알림이 발생하였고 미리보기, 소리 다 정상 동작을 했다..
하지만 모니터를 빼고 사용할 수 없는 노릇이라 이것저것 디스플레이 관련
설정을 찾아보다가 답을 찾아냈다.

“디스플레이를 미러링하거나 공유할 때” 바로 이 설정이 안되어있었다.
어이 없게도 그렇게 시스템 알림 설정을 확인할 때 시선을 조금만 내려봤더라면
바로 해결될 수 있던 문제였는데 한참을 헤매고서야 찾을 수 있었다 ..

당시에는 Monterey 버전을 사용하고 있었고 해당 설정 값이 있었는데
현재 Ventura 버전에서는 설정이 빠진건지, 아니면 외장모니터 연결을 안해서인지
시스템 설정에서 보이지 않는다.
