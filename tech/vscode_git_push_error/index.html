개요

블로그 환경을 Hugo로 변경한 후 블로그 작성 툴로 vscode를 사용하고 있다.
hugo도 지원되고 원격 접속 등 다양한 확장 프로그램이 지원되고 무엇보다 가벼워서 사용 중이다.

한참 블로그 세팅을 변경하고 하면서 긴 시간 터미널로 서버에 접속해 작업하다 ‘git push’를
사용했더니 ‘Missing or invalid credentials.’ 오류가 발생하면서 push가 되질 않았다.
분명 방금 전까지 잘 동작했었는데 .. 이상해서 검색을 해보았는데 생각보다 단순하게 해결이 되었다.

내용

누가봐도 자격 증명에 문제가 생겼고 귀찮은 일이 생길 것 같아 보이지만 ..
그냥… 터미널 새로 열어서 다시 접속하면 해결이 된다..
다양한 해결방법이 더 있는 것으로 보이긴 하지만 해결이 되었기 떄문에.. 끝
